# Simple User Authentication Specification

Build a basic user authentication system with the following features:

## Core Functionality
- User registration with email and password
- User login with JWT token generation
- Password reset flow via email
- Basic profile management

## Technical Requirements
- Use PostgreSQL for user data storage
- Implement bcrypt for password hashing
- JWT tokens with 24-hour expiration
- Email verification required before login

## Security
- Rate limiting on login attempts
- HTTPS required
- Password complexity requirements
- Session management

This is a straightforward specification designed to test Shannon's analysis capabilities.
